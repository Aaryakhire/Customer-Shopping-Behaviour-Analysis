# Customer-Shopping-Behaviour-Analysis

## Description
Customer Shopping Behavior Analysis is an end-to-end data analytics project that combines Python, SQL, and Power BI to study purchase trends, customer segments, and sales performance. The dashboard highlights revenue by category, average spend by age group, discount and promo usage, preferred payment methods, and other actionable insights to support retail decision-making.

---

## Tools and Technologies Used
- MySQL
- Python
- Pandas
- Power BI

---

## Key Tasks Performed
- Collected and cleaned raw shopping data.
- Renamed columns and standardized field names.
- reated derived fields like age groups and purchase frequency scores.
- Performed exploratory data analysis to find patterns and trends.
- Calculated KPIs such as revenue, average rating, and discount usage.
- Designed Power BI visuals and dashboards.
- Compared performance across category, age group, season, and payment method.
